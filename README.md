# SEMAFORO-WOKWI-
Este projeto, que foi realizado no Wokwi, é basicamente um semafóro, ultilizando 3 pushbuttom, 3 resistor, 3 LED's, 1 Halfbreadboard. Funciona da seguinte maneira: 

Botão 1 (vermelho): irá funcionar como um semáforo. O LED 1 ficará ligado por 3 segundos, o LED 2 ficará ligado por 5 segundos e o LED 3 ficará ligado por 1 segundo. Enquanto um LED estiver aceso, os outros deverão permanecer apagados.

Botão 2(amarelo): ao ser pressionado, executará a sequência que já trabalhamos em sala. Os LEDs acenderão com um intervalo de 0,5 segundo entre cada um.

Botão 3(verde): ao ser pressionado, todos os LEDs acenderão juntos, permanecerão ligados por 2 segundos e, em seguida, serão apagados por 1 segundo. Essa sequência deverá se repeti
